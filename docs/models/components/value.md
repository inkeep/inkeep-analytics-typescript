# Value


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 1284.03;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `components.Four[]`

```typescript
const value: components.Four[] = [
  2106.76,
];
```
